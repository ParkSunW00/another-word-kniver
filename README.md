# 이세계 기사 

UnityVersion: **Unity 2022.3.33f**


## 플레이 방식 
1. 머리 위에 표식이 있는 사람을 방향키(⭠,⭡,⭢)를 사용하여 이세계로 보낸다.
2. 이때, 머리 위에 표식이 없는 사람을 이세계로 보내서는 안된다. 잘 피하도록 하자.
